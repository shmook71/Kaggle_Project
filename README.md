# ✈️ Riyadh Air Traffic Intelligence Dashboard

An interactive aviation analytics dashboard built using **Python + Streamlit** to analyze air traffic activity, airline performance, aircraft distribution, and operational risk indicators.

🔗 Live App:
https://kaggleproject-lnwlsh7goky59ejeiqxi.streamlit.app

---

## 🚀 Project Overview

This dashboard provides real-time interactive insights into flight operations including:

- Total flight activity
- Airline distribution
- Aircraft model trends
- Status distribution
- Hourly traffic patterns
- Top airlines & aircraft models
- Operational risk probability scoring

The project transforms raw flight-level data into a clean executive-style analytics dashboard.

---

## 🧠 Key Features

- Dynamic filters (Airline, Status, Flight Type, Date Range)
- KPI summary cards
- Distribution analysis
- Interactive charts
- Dark SaaS-style UI
- ML-based disruption probability scoring

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn

---

## 📊 Dataset

Flight-level structured dataset including:

- Airline information
- Aircraft model
- Origin & destination airports
- Flight time (UTC & Local)
- Operational status
- Flight type
